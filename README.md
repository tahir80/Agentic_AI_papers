# Agentic_AI_papers

A curated log of recent papers on **LLM-based agentic AI** — systems where a large language model is a central component, is given a goal, and can plan and take multi-step actions (tool/API calls, code execution, browsing, environment interaction, memory use, multi-agent coordination, etc.) rather than just producing a single text response.

Each entry below is paired with an accessible blog-post-style summary in [`blog-posts/`](blog-posts/).
